Architecture

A pnpm monorepo with a web dashboard, a mobile client, an API relay/server, and shared libraries.

emergency-mesh

React web dashboard (Vite). Local multi-tab mesh simulation via BroadcastChannel.

artifacts/emergency-mesh

mesh-mobile

Expo mobile client. Uses a WebSocket relay and caches posts for offline viewing.

artifacts/mesh-mobile

api-server

Express + ws relay for real-time coordination and message fanout.

artifacts/api-server

shared libs

DB schema, OpenAPI spec, generated client, and shared validation/types.

lib/*

Stack

  • Frontend: React + Vite (web), Expo (mobile)
  • Backend: Node.js + Express + WebSockets (ws)
  • Data: Drizzle ORM + shared Zod schemas + generated API client
  • Tooling: TypeScript + pnpm workspaces

Setup

Requirements: Node.js (v18+) and pnpm.

Install

git clone <repository-url>
cd mesh-network
pnpm install

Run

pnpm --filter @workspace/api-server run dev
pnpm --filter @workspace/emergency-mesh run dev
pnpm --filter @workspace/mesh-mobile run dev

Modify / Improve

  • DB models: lib/db/src/schema/
  • API contract: lib/api-spec/openapi.yaml
  • API server routes: artifacts/api-server/src/routes/
  • Validation/types: lib/api-zod

See the repo README for deeper technical details and contribution notes.
